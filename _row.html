基本 layout 說明

基本橫列容器：＜div class="row"＞＜/div＞，此為單欄（全寬）容器。

"row" 之後接的，以 cx 開頭的 class name 代表此橫列可以容納幾個直欄，以及直欄寬度的組合方式，例如＜div class="row cx2"＞、＜div class="row cx2a"＞、＜div class="row cx2b"＞都是兩欄的容器，但有不同的寬度組合，請參考下方示範。

＜div class="row cx3"＞、＜div class="row cx3a"＞、＜div class="row cx3b"＞、＜div class="row cx3c"＞都是三欄的容器，但有不同的寬度組合；＜div class="row cx4"＞是等寬四欄容器，請參考下方示範及說明。

＜div class="row"＞如果有 class name "gutter"，則此容器中的欄有欄間距。例如＜div class="row cx3 gutter"＞

上稿者必須注意使用正確的欄寬組合方式。

注意：說明文字中，包html標籤的大於＞、小於＜，以全形符號替代。

下方橫列的粗黑水平線和直欄的細灰框線，是為示範說明方便，在實際應用時不會有。
